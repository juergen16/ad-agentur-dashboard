# Ad Agentur Dashboard

Automatisches Zahlungsstatus Dashboard für Ad Agentur GmbH.

## Automatisches Update
- **Zeit:** Täglich um 08:00 Uhr (MESZ)
- **Datenquelle:** Google Sheets (Zahlungsstatus)
- **Letztes Update: $(date '+%Y-%m-%d %H:%M')

## Manuelles Update
Via GitHub Actions: Actions → Daily Dashboard Update → Run workflow
